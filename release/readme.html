Instructions:

[1] Run `stefann`. Installation details can be found here (requires internet connection).

[2] After application is loaded:
    - Click `Open Image` to get started
    - Click `About Project` to go to the project homepage (requires internet connection)
    - Click `README` to view readme.html
    - Click `Close` to quit application

[3] After an image is loaded, use the following key bindings to perform an operation:
     __________________________________________________________
    |           |                                              |
    | KEY       | OPERATION                                    |
    |___________|______________________________________________|
    |           |                                              |
    | +         | Scale up image                               |
    |___________|______________________________________________|
    |           |                                              |
    | -         | Scale down image                             |
    |___________|______________________________________________|
    |           |                                              |
    | R         | Reset scale of image                         |
    |___________|______________________________________________|
    |           |                                              |
    | G         | Toggle grid                                  |
    |___________|______________________________________________|
    |           |                                              |
    | L MOUSE   | Select region points                         |
    |___________|______________________________________________|
    |           |                                              |
    | R MOUSE   | Remove region points                         |
    |___________|______________________________________________|
    |           |                                              |
    | ESC       | Cancel operation                             |
    |___________|______________________________________________|
    |           |                                              |
    | ENTER     | Commit changes and perform binarization      |
    |___________|______________________________________________|


[4] After initial binarization, use the following key bindings to perform an operation:
     __________________________________________________________
    |           |                                              |
    | KEY       | OPERATION                                    |
    |___________|______________________________________________|
    |           |                                              |

    |___________|______________________________________________|
    |           |                                              |
    | -         | Decrease threshold for binarization          |
    |___________|______________________________________________|
    |           |                                              |
    | TAB       | Toggle inverse binarization                  |
    |___________|______________________________________________|
    |           |                                              |
    | *         | Increase minimum allowed contour area        |
    |___________|______________________________________________|
    |           |                                              |
    | /         | Decrease minimum allowed contour area        |
    |___________|______________________________________________|
    |           |                                              |
    | SPACE     | Select a contour to be edited                |
    |___________|______________________________________________|
    |           |                                              |
    | A-Z       | Insert a character into selected contour     |
    |___________|______________________________________________|
    |           |                                              |
    | BACKSPACE | Undo edit and reset image                    |
    |___________|______________________________________________|
    |           |                                              |
    | ESC       | Cancel operation                             |
    |___________|______________________________________________|
    |           |                                              |
    | ENTER     | Commit changes and save edited image         |
    |___________|______________________________________________|
